Title Twin Cities 7 County ZIP Codes
Abstract This layer defines the boundaries of United States Postal Service (USPS) ZIP code delivery areas. These ZIP code areas are based on actual delivery areas used by mail carriers and geographically defined by Metro GIS parcel boundaries. Section 2 Data Quality Information Top of full metadata Top of page
Attribute Accuracy Attributes for all ZIP codes have been checked and are accurate as of the Currentness Reference date.
Logical Consistency The dataset is topologically 'clean'.
Completeness Entire seven county geographic area is covered.
Horizontal Positional Accuracy The positional accuracy of the dataset has not been tested under the National Standards for Spatial Data Accuracy.
Lineage Line work for the ZIP codes layer is derived primarily from the parcel layers for each county. Each polygon was drawn using heads up digitizing. In areas of ambiguity, due to non residential parcels, additional data sources were used to derive the line work. This included the TLG street centerline data, the 2003 FSA Ortho-imagery, and the current Metropolitan Council city and township boundaries.

Parcel data from April 2009 was downloaded and appended into a single file. This file was geocoded utilizing two geocoders, MapMarker v14 and ZP4. The parcel addresses which matched and met the USPS designation of Delivery Point Validation (DPV) became the source for the ZIP polygons. DPV addresses by definition return a 9 digit ZIP code. A choropleth map was created using all DPV addresses and their 5 digit ZIP code. This provided a visual representation of the actual 5 digit ZIP code delivery area used by the USPS. The heads up digitizing was completed over this color coded map.
